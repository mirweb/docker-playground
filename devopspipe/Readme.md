# Devops Pipeline

## Information

Common devops pipeline with 
* [gitea](https://gitea.com) as git repository
* [nginx](http://nginx.org) as simple reverse proxy

## Running & Maintaining

For starting services 

```bash
docker-compose up -d
```
Following standard logs

```bash
docker-compose logs -f
```
